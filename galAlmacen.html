SISTEMA DE CONTROL DE ALMACÉN
Sistema desarrollado principalmente con Php, Laravel, MySQL y jQuery.   Contiene el inventario del almacén y administra la información de entradas y salidas de material y herramientas.

Sincroniza sus bases de datos con el sistema de nóminas para conocer si el personal estaba autorizado para el préstamo de equipo.
