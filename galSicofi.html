SICOFI SISTEMA DE CONTROL FINANCIERO
Sicofi fue desarrollado inicialmente con PHP y MySQL.   En sus inicios solo contenía una base de datos de ingresos y egresos, mas tarde se añadieron funcionalidades como: Solicitudes de Fondos, Gasto por proyectos y capítulos, elaboración de formatos de cheques y pólizas, reportes por proyecto, avisos por correo, etc.

Al ser mi primer sistema profesional contiene diseño y estructura simple, pero mas tarde se integraron mas librerías y el uso de javascript para vistas mas dinámicas, recibe alrededor de 5000 consultas mensuales y es el sistema mas importante de la administración.
